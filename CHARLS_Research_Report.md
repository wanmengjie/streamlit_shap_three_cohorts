# CHARLS 老年抑郁与认知共病：前瞻性因果机器学习研究报告

## 1. 研究概览 (Research Summary)
- **数据来源**: 2011-2020 年中国健康与养老追踪调查 (CHARLS)。
- **研究设计**: 前瞻性滚动入射队列 (Prospective Rolling-wave Incident Cohort)。
- **最终样本量**: **10,207** 条深度随访记录（起点必须为认知健康）。
- **核心结论**: 抑郁状态对未来认知障碍共病发生具有显著的因果推动作用，其**平均因果效应 (ATE) 为 0.0907** (95% CI: 0.082-0.099)。

## 2. 数据处理与筛选流 (Data Flow)
1. **基础过滤**: 筛选年龄 ≥ 60 岁的老年人群。
2. **入射队列定义**: 剔除基线已患认知障碍者（Total Cognition ≤ 10），确保模型捕捉的是从健康到患病的新发过程。
3. **滚动配对**: 采用 Wave T 特征预测 Wave T+1 结局，最大化利用 2011-2020 九年随访信息。
4. **特征严选**: 从 100+ 原始变量中精选 **36 个核心指标**，涵盖人口学、生理、行为及社会保障。
