# CHARLS 数据说明

## 数据来源

中国健康与养老追踪调查（China Health and Retirement Longitudinal Study, CHARLS）多波次纵向数据。

## 变量定义

| 变量 | 定义 | 编码/单位 |
|------|------|-----------|
| ID | 个体唯一标识 | — |
| wave | 调查波次 | 1, 2, 3, 4 |
| age | 年龄 | 岁 |
| cesd10 | CES-D-10 抑郁量表得分 | 0–30，≥10 定义为抑郁 |
| total_cognition / total_cog | 认知总分 | ≤10 定义为认知受损 |
| is_comorbidity | 抑郁-认知共病 | 0/1（抑郁且认知受损同时存在） |
| is_comorbidity_next | 下一波共病 | 0/1 |
| exercise | 规律运动 | 0/1 |
| sleep | 睡眠时长 | 小时/天 |
| smokev | 当前吸烟 | 0/1 |
| drinkev | 当前饮酒 | 0/1 |
| rural | 农村居住 | 0/1 |
| gender | 性别 | 0/1 |
| edu | 教育水平 | 1–4 级 |
| marry | 婚姻状况 | 0=未婚/离异/丧偶，1=在婚 |
| bmi | 体质指数 | kg/m² |
| chronic_burden | 慢性病负担 | 数量 |

## 数据筛选步骤（对应表 1）

1. 原始记录（人-波）
2. 年龄 ≥ 60 岁
3. CES-D-10 非缺失
4. 认知得分非缺失
5. 下一波共病非缺失
6. 基线无共病、既往无共病（入射队列）→ 14,386

## 三轴线定义

| 轴线 | 定义 |
|------|------|
| A（健康） | 基线无抑郁、无认知受损 |
| B（仅抑郁） | 基线抑郁阳性、认知正常 |
| C（仅认知受损） | 基线认知受损、抑郁阴性 |

## 缺失值处理

- 完整病例分析：仅保留运动无缺失者
- 建模阶段：连续变量中位数插补，分类变量众数插补
