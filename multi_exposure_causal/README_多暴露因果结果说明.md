# 多暴露因果分析结果说明

## 分析设计

对 B 轴（抑郁基线）和 C 轴（认知受损基线）分别估计以下 5 个暴露对 incident 抑郁–认知共病的因果效应（ATE，风险差）：

| 暴露 | 定义 | 预期方向 |
|------|------|----------|
| exercise | 规律运动 1 vs 0 | 保护（负） |
| sleep_adequate | 充足睡眠(≥6h) 1 vs 0 | 保护（负） |
| smokev | 当前吸烟 1 vs 0 | 风险（正） |
| drinkev | 当前饮酒 1 vs 0 | 不确定 |
| is_socially_isolated | 社会隔离 1 vs 0 | 风险（正） |

## 主要结果

**所有暴露的 95% CI 均包含 0，无 statistically significant 结果。**

| 暴露 | 轴线 | ATE | 95% CI | 显著? |
|------|------|-----|--------|-------|
| exercise | B | −0.040 | (−0.137, 0.057) | 否 |
| exercise | C | +0.032 | (−0.077, 0.141) | 否 |
| sleep_adequate | B | −0.016 | (−0.111, 0.080) | 否 |
| sleep_adequate | C | −0.011 | (−0.127, 0.105) | 否 |
| smokev | B | +0.020 | (−0.102, 0.142) | 否 |
| smokev | C | +0.010 | (−0.125, 0.146) | 否 |
| drinkev | B | +0.016 | (−0.089, 0.120) | 否 |
| drinkev | C | +0.033 | (−0.094, 0.159) | 否 |
| is_socially_isolated | B | 10.60 | (−60.5, 81.7) | **不可靠** |
| is_socially_isolated | C | 5.34 | (−34.3, 44.9) | **不可靠** |

## 说明

1. **is_socially_isolated**：ATE 超出风险差合理范围 [−1, 1]，估计不可靠，可能因社会隔离阳性率低（约 7–8%）导致 Causal Forest 不稳定。
2. **方向性**：exercise、sleep_adequate 在 B 轴为负（保护方向），smokev、drinkev 为正（风险方向），与既往文献一致，但均未达统计学显著。
3. **建议**：可考虑增大样本、使用更精细的暴露定义，或做亚组分析以探索异质性。
