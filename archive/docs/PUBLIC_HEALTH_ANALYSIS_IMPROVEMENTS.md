# 公共卫生视角：数据分析可提升点

**视角**：公共卫生/流行病学数据分析规范（STROBE、因果推断、可重复性）。  
**目标**：在现有管线基础上，从方法学、报告规范、稳健性三方面提出可落地提升建议。

---

## 一、方法学与定义

### 1.1 结局与暴露的标准化定义与引用

- **抑郁**：当前使用 CES-D-10 ≥ 10。建议在方法中明确写出“采用 CES-D-10 截断值 10 定义抑郁症状阳性”，并引用量表来源与国内/国际常用截断值（如 Andresen 等或 CHARLS 手册）。
- **认知受损**：当前使用 total_cognition ≤ 10。需明确该变量对应 CHARLS 中哪一认知模块（如 TICS、画图、回忆等）、截断值 10 的依据（手册或文献），避免审稿人质疑“为何是 10”。
- **共病**：同时满足抑郁+认知受损，定义清晰。建议在文中用“depression–cognition comorbidity”或“incident depression–cognitive impairment comorbidity”统一表述。
- **运动（exercise）**：因果分析中作为二值干预（fillna(0).astype(int)）。需在方法中明确“运动”在 CHARLS 中的操作定义（如“过去一个月是否规律运动/锻炼”及编码 0/1），缺失记为 0 的处理会带来**选择/信息偏倚**风险，建议在敏感性分析或讨论中说明。

**建议**：在代码或文档中增加 `ANALYSIS_DEFINITIONS.md`，列出上述定义与对应列名、截断值、文献，便于方法与代码一致。

---

### 1.2 因果推断前提的显式说明

- **可交换性**：当前通过 Causal Forest DML 控制 X 中混杂。建议在方法中列出“最小充分混杂集”或至少列出主要混杂因素（年龄、性别、教育、城乡、基线慢性病、基线功能等），并说明“假定在给定 X 下治疗可交换”。
- **正性**：未检查“各协变量组合下均有 T=0 与 T=1”。建议增加简单诊断：如按十分位或分类后，检查是否存在某层内仅有 T=0 或 T=1；若存在，在文中说明并考虑合并层或限制分析人群。
- **因果图**：建议在补充材料中提供简化 DAG（暴露 → 结局；混杂 → 暴露/结局），标明数据中测量的变量，便于审稿人理解假设。

---

### 1.3 干预变量缺失的处理

- 当前：`T_series = df_sub[T].fillna(0).astype(int)`，将运动缺失全部视为“未运动”。若缺失并非随机（MNAR），会低估或高估 ATE。
- **建议**：  
  - 主分析保持现状时，在方法中写明“将干预缺失视为 0（未暴露）”。  
  - 敏感性分析：限制在干预变量无缺失的样本中重新估计 ATE，或对干预做多重插补后比较，并在讨论中说明局限性。

---

## 二、缺失数据与选择（流失）

### 2.1 流失/纳入流程图（STROBE）

- 当前未在结果中系统报告“每一步排除后剩余 n”。
- **建议**：在预处理中记录并输出**流失表**（如：原始观测数 → 年龄≥60 → 有 CES-D-10 → 有认知分 → 有下一波共病结局 → 基线非共病 → 最终分析样本），写入 `LIU_JUE_STRATEGIC_SUMMARY/attrition_flow.csv`（或等同路径），并在论文中提供流程图（Consort/STROBE 风格），标注每步 n 与排除原因。

**落地**：已在预处理中增加按步骤记录 n，并写入 `preprocessed_data/attrition_flow.csv`；主流程将其复制到汇总目录，便于直接用于图表与正文。

### 2.2 缺失数据报告（Table 1 补充）

- 当前：关键变量（CES-D-10、认知、下一波结局）采用 listwise deletion；其余特征在建模时用中位数插补。
- **建议**：  
  - 在 Table 1 或补充表中增加“缺失比例”列（每变量 % missing），便于评估选择偏倚风险。  
  - 方法中明确：“基线抑郁/认知/结局缺失者被排除；其余协变量缺失采用中位数插补”，并讨论对因果估计的可能影响（如缺失较多时考虑多重插补的敏感性分析）。

## 三、报告与可重复性

### 3.1 模型性能的报告

- 当前已报告 AUC 及 Bootstrap 95% CI，符合规范。建议同时报告：  
  - **灵敏度/特异度**（或 Youden 最优截断下的值），便于临床解读。  
  - **校准**：已有校准曲线；建议在文中或补充材料中报告 Brier score 或校准截面的数值摘要（如按十分位）。  
  - **亚组**：亚组分析中 AUC 的 95% CI（若样本量允许），避免仅点估计。

### 3.2 因果效应报告

- ATE 及 95% CI 已输出。建议补充：  
  - **效应尺度**：ATE 为风险差（RD）还是其他？若 Y 为 0/1，当前 EconML 输出应为 RD 或近似 RD，需在方法中写明。  
  - **亚组 CATE**：城乡、年龄组、性别的 CATE 已有；建议在文中用表格或森林图呈现“点估计 + 95% CI”，并注明为探索性、不做多重比较校正。

### 3.3 多重比较与探索性

- 当前对多组发病率、多模型比较、多亚组 CATE 未做多重比较校正。  
- **建议**：在方法或图注中明确“亚组与多模型比较为探索性分析，未进行多重比较校正”；若需确认性结论，可对关键比较（如 B vs C 的 ATE）预先指定并考虑 FDR/Bonferroni。

### 3.4 临床决策支持与 DCA

- 当前决策支持已基于“增加运动/改善睡眠”的反事实，与因果设定一致。建议在文中说明：  
  - 决策曲线分析（DCA）的净获益如何解读（阈值范围、与“全干预/全不干预”的比较）。  
  - 反事实情景（如“若增加运动”）的假设：在给定模型下外推，非 RCT 证据，需在讨论中限定解读范围。

## 四、稳健性与敏感性分析

### 4.1 结局/暴露定义的敏感性

- **抑郁截断值**：如 CES-D-10 用 8 或 12 替代 10，发病率与模型性能如何变化（可做简要表格）。  
- **认知截断值**：同理，若文献支持其他截断值，可做 1 次敏感性分析。  
- **运动定义**：若数据允许，比较“二值 vs 有序/连续”运动变量的因果估计差异。

### 4.2 缺失与插补

- **完整病例**：限制在关键变量无缺失的样本，重新跑因果与预测，与主分析对比。  
- **多重插补**：对协变量（或含 T）做 MICE 等插补，在插补数据上重新估计 ATE 并 Rubin 合并，作为敏感性分析（可选，计算量较大）。

### 4.3 人群与时间

- **年龄**：已按 65/75 分组；可补充“仅 65–80 岁”或“仅 75+”的亚组或敏感性分析，说明结果在不同年龄段的稳健性。  
- **波次**：若有多波，可说明主分析基于哪几波、是否包含所有波次，以及是否检查过“波次”与结局/暴露的交互。

---

## 五、泛化性与伦理

### 5.1 泛化性

- 明确写出：本研究基于 CHARLS，对象为中国 60 岁及以上社区老年人；结果主要适用于同人群、同定义下的预测与因果解释，外推至其他地区/年龄需谨慎。

### 5.2 伦理与数据

- 文中注明：CHARLS 已获相应伦理审批及知情同意；分析使用公开/授权数据，符合伦理要求。  
- 代码与（在许可范围内）分析流程的可重复性：当前已有固定种子与模块化脚本；建议在补充材料中提供“运行环境与依赖版本”，便于复现。

---

## 六、小结与优先顺序

| 优先级 | 类别       | 建议 |
|--------|------------|------|
| **高** | 报告       | 增加流失流程图数据（每步 n）并写入汇总目录；方法中明确结局/暴露/运动定义及引用。 |
| **高** | 因果       | 方法中写出“最小充分混杂集”与正性假设；干预缺失的处理及敏感性说明。 |
| **中** | 缺失与偏倚 | Table 1 或补充表报告主要变量缺失比例；讨论 listwise deletion 对结果的可能影响。 |
| **中** | 稳健性     | 至少 1 次敏感性分析（如抑郁/认知截断值或完整病例）。 |
| **低** | 探索性     | 亚组 CATE 的 95% CI；多重比较与探索性结论的明确标注；泛化性与伦理一句说明。 |

以上建议在不大幅改动现有代码的前提下，通过补充文档、预处理流失记录、方法/讨论中的文字说明即可显著提升论文在公共卫生与流行病学审稿中的规范性与说服力。
