# 讨论部分草稿（供论文引用）

## 4.1 轴线 C 预测性能差异原因分析

轴线 C（仅认知受损人群）的 AUC（约 0.61）显著低于轴线 A（约 0.73）和 B（约 0.70），可能原因包括：

1. **样本量较小**：认知受损亚组样本量有限，模型学习不充分。
2. **基线认知受损**：该人群异质性更大，特征变异性高，预测难度增加。
3. **关键预测特征缺失**：认知域细分（记忆、执行功能等）在建模时被排除以防泄露，可能损失有用信息。
4. **疾病机制**：抑郁-认知共病在认知受损人群中可能呈现不同轨迹，单一预测模型难以捕捉。

**优化建议**：增加认知亚组样本、纳入认知保留的代理指标、考虑迁移学习或集成策略。

## 4.2 运动在轴线 C 的 ATE 为正的探索

运动在轴线 C 中 ATE 为正（即运动与共病风险增加相关）与预期不符，可能原因：

1. **反向因果**：认知受损人群中，共病早期症状可能导致运动减少，而“仍坚持运动”者可能为病情较轻者，产生选择偏倚。
2. **运动强度不当**：该人群若进行高强度运动，可能加重负担，需按运动类型（轻/中/重）分层分析。
3. **残余混杂**：认知功能相关协变量（如认知域得分、痴呆家族史）未充分纳入。

**建议**：开展按运动强度分层分析、补充认知相关协变量重估、谨慎解读该亚组结果。

## 4.3 临床转化方案

1. **筛查流程**：基于 DCA 最优阈值（见 calibration_brier_report.txt），对 60 岁及以上老年人进行高危筛查；建议基线健康人群每 2 年、抑郁/认知受损人群每年筛查。
2. **分层干预**：仅抑郁人群优先推荐规律运动；健康人群重点开展睡眠与 BMI 管理；认知受损人群需个体化评估后制定运动方案。
3. **社会隔离**：估计不稳定，建议未来研究扩大样本、采用更精细的暴露测量（如隔离持续时间/程度）。
