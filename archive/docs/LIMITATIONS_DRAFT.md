# 研究局限性（供论文讨论部分引用）

## 一、数据与测量

1. **干预暴露为自报变量**：运动、睡眠等均为问卷自报，存在回忆偏倚与测量误差，可能低估或高估真实暴露水平。
2. **未考虑共病发生时间**：结局为二分类（是否发生），未纳入发生时间与随访时长的交互效应。
3. **未纳入潜在混杂**：遗传、肠道菌群、药物使用等未纳入，可能存在残余混杂。

## 二、方法学

4. **因果推断假设**：PSM/PSW/DML 均依赖可忽略性假设，未测混杂可能影响结论。
5. **E-Value 解读**：E-Value 量化未测混杂需达到的强度才能推翻结论，但不能证明无混杂。
6. **社会隔离等低样本量**：治疗组样本极少时，ATE 估计不稳定，已采用贝叶斯方法补充。

## 三、泛化性

7. **内部验证为主**：跨数据集验证（如 CLHLS）需数据支持，可列为未来研究。
8. **人群限定**：60 岁及以上中国老年人，外推至其他人群需谨慎。

## 四、解决思路建议

- 测量：采用客观测量（如加速度计、睡眠监测）验证自报暴露
- 时间：采用生存分析（Cox 模型）纳入发生时间
- 混杂：开展孟德尔随机化或工具变量分析补充
- 泛化：获取 CLHLS 等队列数据进行外部验证
