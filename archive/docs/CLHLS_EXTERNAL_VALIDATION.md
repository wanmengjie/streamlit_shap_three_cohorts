# CLHLS 跨数据集外部验证（展望）

## 目的
在 CLHLS（中国老年健康长寿调查）数据上验证本研究的预测模型与因果效应，评估泛化能力。

## 数据要求
- CLHLS 需包含：年龄、性别、居住地、BMI、慢性病、抑郁相关指标（如 CES-D 或类似）、认知相关指标
- 结局：抑郁-认知共病或类似定义
- 至少 2 波次纵向数据

## 实施步骤（待数据获取后）
1. 变量映射：将 CLHLS 变量对齐至 CHARLS 建模特征
2. 模型迁移：将 CHARLS 训练的冠军模型直接预测 CLHLS 验证集
3. 评估指标：AUC、AUPRC、校准曲线、Brier
4. 可选：在 CLHLS 上重新训练并比较

## 占位说明
当前无 CLHLS 数据，本文件为方法学展望。获取数据后可新增 `charls_clhls_validation.py` 实现上述流程。
