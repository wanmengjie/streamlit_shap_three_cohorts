| 项目 | 旧版 | 新版 |
|------|------|------|
| 主因果方法 | Causal Forest DML | **TLearner**（RF nuisance） |
| Nuisance 模型 | CatBoost | RandomForest |
| 阈值确定 | test set | **Internal CV (training), Youden** |
| Bootstrap 次数 | 500 | 1000（Stratified） |
| 已移除变量 | — | smokev, rgrip, grip_strength_avg, psyche, sleep_adequate |
| 干预数量 | 5（含 smoking, adequate sleep） | **5**（exercise, drinkev, is_socially_isolated, bmi_normal, chronic_low） |
| PSM 卡尺 | 0.2 SD | **0.024**（PS 标准差倍数） |
| PSW 截断 | — | [0.1, 50] |
| Overlap trimming | — | PS ∈ [0.05, 0.95] |
