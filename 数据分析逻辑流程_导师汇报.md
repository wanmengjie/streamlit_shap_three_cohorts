# 老年抑郁-认知共病预测与因果分析：数据分析逻辑流程

**汇报用 | 基于 CHARLS 的因果机器学习框架**

---

## 一、整体框架（一句话）

> 在 CHARLS 纵向数据上，按基线状态将老年人分为三组（健康 / 仅抑郁 / 仅认知受损），分别进行**共病风险预测 → 可解释性归因 → 因果效应估计 → 临床决策支持**，形成「预测—解释—因果—应用」的完整链条。

---

## 二、研究设计核心

| 项目 | 定义 |
|------|------|
| **结局** | `is_comorbidity_next`：下一波是否发生抑郁-认知共病（二分类） |
| **主干预** | `exercise`：是否规律运动（≥1次/周） |
| **人群** | 60 岁及以上，基线无共病 |
| **三队列** | **A** 健康基线 / **B** 仅抑郁基线 / **C** 仅认知受损基线 |
| **截断值** | 抑郁：CES-D-10 ≥10；认知受损：认知总分 ≤10 |

---

## 三、数据流与主流程

```
CHARLS 原始数据 (CHARLS.csv)
        ↓
[1] 预处理：年龄筛选、缺失处理、队列定义、baseline_group 标记
        ↓
df_clean（三队列合并）
        ↓
按 baseline_group 拆分为 df_a / df_b / df_c
        ↓
[2] 每条队列 Cohort A/B/C 执行相同分析协议 run_cohort_protocol（旧称 run_axis_protocol）
        ↓
[3] 汇总 + 敏感性 + 多暴露 + 方法交叉验证
        ↓
results/tables、results/figures、results/models
```

## 四、单队列分析协议（run_cohort_protocol）

每条轴线（A/B/C）依次执行以下步骤：

### Step 1：预测打擂（01_prediction）

- **目标**：预测下一波是否发生共病
- **方法**：15 个模型（LR、RF、XGB、LGBM、CatBoost、SVM、MLP 等）RandomizedSearchCV 调参
- **数据划分**：GroupShuffleSplit（按 ID 分组，80% 训练 / 20% 测试），避免同一人跨波泄露
- **预处理**：Imputer + StandardScaler 全部在 Pipeline 内，仅用训练折 fit
- **阈值**：在 development set 内用 Internal CV 确定最优阈值，测试集使用固定阈值
- **输出**：冠军模型、model_performance_full_*.csv、roc_data.json、雷达图

### Step 2：SHAP 可解释性（02_shap）

- **目标**：解释冠军模型，找出对预测贡献最大的特征
- **方法**：SHAP 归因 + 分层 SHAP + 交互 SHAP
- **意义**：若可干预因素（运动、睡眠等）排名靠前，为后续因果分析提供依据

### Step 3：因果分析（03_causal）

- **目标**：估计运动对共病风险的因果效应（ATE）
- **方法**：Causal Forest DML（EconML）
  - Nuisance 模型：Random Forest 拟合 E[Y|X]、E[T|X]
  - 五折交叉拟合，1000 棵因果树，honest 估计
- **假设**：无混淆、重叠、SUTVA
- **输出**：ATE、95% CI、ITE、倾向评分重叠图、E-value

### Step 4：敏感性分析（07_sensitivity）

- **目标**：检验因果估计稳健性
- **内容**：特征噪声、E-value、未测量混杂模拟

### Step 5：临床评价与决策支持（04_eval、05_decision）

- **04_eval**：校准曲线、Brier 分解、AUPRC
- **05_decision**：DCA 决策曲线、反事实模拟（若增加运动，风险如何变化）

### Step 6：外部验证（04b_external_validation）

- **时间验证**：wave<4 训练，wave4 验证
- **区域验证**：东中部训练，西部验证

### Step 7：剂量反应（04c_dose_response）

- **方法**：RCS 限制性立方样条，探索运动强度与结局的非线性关系

### Step 8：亚组分析（06_subgroup）

- **分层**：居住地、年龄、性别、教育、慢性病、自评健康
- **输出**：各亚组 AUC、CATE、样本量

## 五、全流程汇总分析（main 后半段）

| 模块 | 内容 |
|------|------|
| **截断值敏感性** | CES-D≥8/10/12、认知≤8/10/12 共 9 种组合，重训模型、重估 ATE |
| **多暴露因果** | 运动、睡眠、吸烟、饮酒、社会隔离 5 个暴露分别估计 ATE |
| **扩展干预分析** | 加入 BMI 正常、慢性病负担低等，生成森林图 |
| **PSM/PSW 交叉验证** | DML vs PSM vs PSW 三种方法对比，审稿 P1 |
| **低样本优化** | 社会隔离、慢性病负担低等小样本暴露的稳健估计 |
| **插补敏感性** | Median/Mean/Mode/KNN/MICE 对预测 AUC 的影响 |

## 六、方法学要点（防审稿质疑）

| 问题 | 处理方式 |
|------|----------|
| **数据泄露** | 所有预处理在 Pipeline 内；GroupShuffleSplit 按 ID 分组 |
| **阈值寻优** | 在 development set 内用 Internal CV 确定，测试集固定阈值 |
| **IPW 极端权重** | PS 限制 [0.01, 0.99]，权重限制 [0.1, 50] |
| **Bootstrap** | 分层 Bootstrap，n=1000 |
| **重叠假设** | 倾向评分分布图（暴露组 vs 对照组） |
| **未测量混杂** | E-value 报告 |
